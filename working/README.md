# Working Folder

Create one folder per review case:

```text
working/CASE-YYYYMMDD-short-title/
├── raw/
├── input-register.md
├── missing-inputs.md
├── extracted/
├── invention/
├── creative/
├── evidence/
├── score_cards/
├── council/
└── reports/
```

Do not overwrite original raw files. Record low-confidence extraction and missing information explicitly.
